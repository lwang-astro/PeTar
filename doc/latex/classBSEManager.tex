\doxysubsection{Detailed Description}
S\+S\+E/\+B\+SE interface manager. 

The class provides the interface to call single and binary stellar evolution (evolve\+Star and evolve\+Binary); and also the time step estimators (get\+Time\+Step\+Star, get\+Time\+Step\+Binary).
